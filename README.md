## ⚙️ Tech Used

- React
- Vite
- TypeScript
- Sass

## 🛠️ Getting Started

Make sure you have the following installed:
```
Node.js >= 16
npm or yarn
```
Install once the repository is cloned locally:
```
npm install
# or
yarn
```
Run locally:
```
npm run dev
# or
yarn dev
```
The app will be available at: [http://localhost:5173/](http://localhost:5173/)
